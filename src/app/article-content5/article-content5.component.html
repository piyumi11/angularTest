  • PHP 7.2.15-0
  • ubuntu 0.18.04.2
  • MySQL server 5.7.25-0
  • Apache 2.4.29
  • CodeIgniter

Find the github code exmple in here. Git commit add login implmentation contains the complete code.

Step 01 :

Create the database table to insert users. 

          	
			CREATE TABLE users (
			    id INT NOT NULL PRIMARY KEY AUTO_INCREMENT,
			    username VARCHAR(50) NOT NULL UNIQUE,
			    password VARCHAR(255) NOT NULL,
			    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
			);

Step 02 :

Create the controller class : Account_controller.php to process the login, signin and logout functions. This example use password_hash() one way PHP hashing function to create strong password hash within signin route. This applies a salt to password, hence two users have same password this function create different password hashes.

When login, use password_verify() PHP method to compare user given password and the password in database table.

Logout method unset the all session variables and destroy the session.

Step 03 :

Login_form.php and Signin_form.php contains the HTML forms.

Step 04 :

header.php file is common for all HTML pages. This contains the links to sigin, login and logout. Logout link and logged user name display only user has already logged in to the session else display the login link.
