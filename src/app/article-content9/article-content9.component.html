TCPDF is a php library that can use to create pdf documents using html. Integration of TCPDF to CodeIgniter has few simple steps compare to other PDF libraries. Following example relate to CodeIgniter php application StudentApplication in github.

Step 01 :

Download TCPDF library from http://www.tcpdf.org/.

Step 02 :

Create a folder 'tcpdf' inside the 'application/helpers' directory of CodeIgniter application and copy all TCPDF library files into that.

Step 03 :

Create 'pdf_helper.php' file inside 'application/helpers' folder and import TCPDF as below code.


				<?php 		
					function tcpdf()
					{{ '{' }} 
					    require_once('tcpdf/tcpdf.php');
					{{ '}' }}
				?>

Step 04 :

This application create PDF document to display data in a html table. Add a function as below to a php controller class. Read the comments inside the file to understand the TCPDF functions. 


				public function pdf_report() {{ '{' }} 
				
				          $this->load->helper('pdf_helper'); 	// import pdf helper 
				          tcpdf();  		// execute helper function
				          
				          // retrive the student details from database 
				          $query = $this->db->get("stud");
				          $data['records'] = $query->result();
				          $data['page_title'] = "All students list";  
				          
				          $this->load->view('pdf_report', $data);  // view file pdf_report.php
			          
      				{{ '}' }}

Step 05 :

Create a 'pdf_report.php' inside the 'view' folder of the CodeIgniter app and add the code inside this file. This code convert the data return by controller into pdf file using the library TCPDF. Read the documentation to understand the TCPDF functions inside 'pdf_report.php' file. Git commit has the complete code relate to this article.
