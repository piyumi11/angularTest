What is Spring MVC?

Spring MVC framework use to develop web application with decoupling model, view and controller.

What is Spring Boot?

Spring application needs list of configuration files. i.e. dispatcher servlet, component scan, view resolver, resource mappings etc. If the spring application use hibernate, it requires to configure entity manager, transaction manager, data source in separate configuration files.

Spring Boot auto configure the application by looking at available framework on class path and existing configurations without above mentioned configuration files.

Spring boot starters are convenient dependencies which built with basic spring application dependencies like spring web, hibernate validator, jpa, Jackson databind, log4j, tomcat embedded servlet container. For instance, using spring boot starter web dependency, we can develop production ready web application with restfull web services without specifying each dependency above and compatible versions.


	<dependency>
		<groupId>org.springframework.boot</groupId>
		<artifactId>spring-boot-starter-web</artifactId>
	</dependency>
