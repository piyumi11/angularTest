This example used :
  • JDK 1.8
  • Eclipse IDE
  • Apache tomcat server 7
  • Maven

Step 01 :

Create a dynamic web project using eclipse IDE.

Step 02 :

Then convert the created project into maven project. It will create the POM.xml inside the project root directory. Following maven dependencies should be added to POM.xml to use the jersey. 


     <dependencies>
        <dependency>
            <groupId>org.glassfish.jersey.core</groupId>
            <artifactId>jersey-server</artifactId>
            <version>2.20</version>
        </dependency>

        <dependency>
            <groupId>org.glassfish.jersey.containers</groupId>
            <artifactId>jersey-container-servlet</artifactId>
            <version>2.20</version>
        </dependency>
    </dependencies>

Step 03 :

Create the service class with REST service methods


       		import javax.ws.rs.GET;
		import javax.ws.rs.Path;
		import javax.ws.rs.PathParam;
		import javax.ws.rs.core.Response;
		
		@Path("/sayhello")
		public class HelloWorldService {{ '{' }}
		
		    @GET
		    @Path("/{{ '{' }}name{{ '}' }}")
		    public Response sayHello(@PathParam("name") String msg) {{ '{' }}
		        String output = "Hello, " + msg + "!";
		        return Response.status(200).entity(output).build();
		    {{ '}' }}						    
		{{ '}' }}

Step 04 :

Specify the package which reside the service classes implementing jersey ResourceConfig class 

          		 	
    import org.glassfish.jersey.server.ResourceConfig;

	public class HelloWorldApplication extends ResourceConfig {{ '{' }}
	    public HelloWorldApplication() {{ '{' }}
	        packages("com.example.services");
	    {{ '}' }}
	{{ '}' }}

Step 05 :

If the project does not have a web.xml file create as below.


	<?xml version="1.0" encoding="UTF-8"?>
	<web-app xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" 
	xmlns="http://java.sun.com/xml/ns/javaee" 
	xsi:schemaLocation="http://java.sun.com/xml/ns/javaee 
	http://java.sun.com/xml/ns/javaee/web-app_3_0.xsd" 
	version="3.0">
		  <display-name>Web Service Demo</display-name>
		    <servlet>
		        <servlet-name>WebServiceDemo</servlet-name>
		        <servlet-class>org.glassfish.jersey.servlet.ServletContainer
		        </servlet-class>
		        <init-param>
		            <param-name>javax.ws.rs.Application</param-name>
		            <param-value>com.example.app.HelloWorldApplication</param-value>
			     <{{ '!' }} --ResourceConfig class implementation -->
		        </init-param>
		        <load-on-startup>1</load-on-startup>
		    </servlet>
		    <servlet-mapping>
		        <servlet-name> WebServiceDemo </servlet-name>
		        <url-pattern>/*</url-pattern>
		    </servlet-mapping>
	</web-app>
