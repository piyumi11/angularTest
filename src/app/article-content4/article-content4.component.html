This example use LAMP stack and CodeIngiter php framework to develop a small web-site. LAMP : Linux, Apache, MySQL, PHP is a free open source web stack use to host web applications. CodeIgniter is a free php framework with lot of developer convenient features to develop web sites with high performance. Follow the steps provided below to build this website.

Step 01 :

First have a Linux environment. Following commands tried on Ubuntu 18.04(LTS).

Step 02 :

Install apache http server.


      		sudo apt install apache2

Step 03 :

Install mysql server


      		sudo apt install mysql-server

Step 04 :

Install PHP 7.2, apache support and mySql support.


      		sudo apt install php7.2 libapache2-mod-php7.2 php-mysql

Step 05 :

Login to mysql shell and create a user, database for the application.this password should be different to root user password. 


		sudo mysql -u root
		
		CREATE DATABASE webdata;
		GRANT ALL ON webdata.* TO 'webuser' IDENTIFIED BY 'password';			
		
		CREATE TABLE stud (
		roll_no INT(11) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
		name VARCHAR(30),
		created DATETIME DEFAULT NULL,
		modified DATETIME DEFAULT NULL
		);

Step 06 :

Download the CodeIgniter framework zip, extract it and copy all the files into var/www/html directory.

Step 07 :

download the phpExmple and copy into var/www/html/codeigniter/application directory.

Step 08 :

See the output and check how php routes works in this application.
